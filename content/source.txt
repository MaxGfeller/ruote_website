h2. source

Ruote's source code is located at the "github":http://github.com

* "ruote":http://github.com/jmettraux/ruote/ - the core engine

* "ruote-web2":http://github.com/jmettraux/ruote-web2/ - a single-process "Rails":http://rubyonrails.org web application ("online demo":http://difference.openwfe.org:3000/)
* "ruote-rest":http://github.com/jmettraux/ruote-rest/ - a wrapper for ruote based on "Rack":http://rack.rubyforge.org
* "ruote-fluo":http://github.com/jmettraux/ruote-fluo/ - some kind of an browser based editor for process definitions ("online demo":http://difference.openwfe.org:4567/?pdef=pdef.rb)

* "ruote-kit":http://github.com/kennethkalmer/ - coming soon

See also "rufus":http://rufus.rubyforge.org, a set of ruby gems spun off of ruote.
